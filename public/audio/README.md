# Truth ambient audio

Drop a short looping ambient track here named `ambient.mp3`.

Recommended:
- Format: MP3 or M4A, 128 kbps is plenty
- Length: 30 to 90 seconds, designed to loop seamlessly
- Mood: low, atmospheric, slightly mysterious. Think
  underground bar — soft bass, room tone, a hint of muffled
  music behind a door. No vocals.
- Volume: keep peak around -12 dBFS so the toggle's 0.45
  multiplier doesn't sound shrill on top of it.

The AudioToggle component HEAD-checks for `/audio/ambient.mp3`
and only renders the floating speaker button if it exists,
so dropping the file in is enough to enable the feature.

Free royalty-free sources you can grab a track from:
- https://pixabay.com/music/search/ambient/
- https://uppbeat.io/browse/music/cinematic
- https://www.epidemicsound.com/ (paid)
